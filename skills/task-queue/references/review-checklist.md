# 审核检查清单

## 自检清单（提交前必查）

- [ ] 代码无编译错误
- [ ] API 路由符合 /api/v1/ 前缀规范
- [ ] 错误码符合 40001-40099 / 40101-40199 / 40401-40499 / 50001-50099 规范
- [ ] 所有交付物清单项完成
- [ ] 代码有中文注释（用户偏好）
- [ ] 没有硬编码密码或密钥
- [ ] 有日志输出（关键操作）
- [ ] 数据库表前缀为 biz_

## 代码审查清单（Go 后端）

- [ ] 文件名 snake_case，导出类型 PascalCase
- [ ] handler → service → repository 三层调用
- [ ] 错误显式处理，无 panic
- [ ] 路由有分组（/api/v1/...）
- [ ] JWT 中间件保护私有路由
- [ ] 参数校验（参数错误返回 40001）
