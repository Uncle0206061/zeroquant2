# 反馈模板

## 完成反馈卡（DoneCard）

```markdown
# 完成反馈卡 DoneCard-{任务编号}
完成时间：YYYY-MM-DD HH:mm

## 任务编号
TC-XXX-niuniu

## 交付物确认
- [x] 交付物1：路径/说明
- [x] 交付物2：路径/说明

## 自检结果
[ ] 代码无编译错误
[ ] API 路由符合规范
[ ] 所有交付物清单项完成
...

## 遗留问题
[如有]

## 耗时
X 小时
```

## 阻塞卡（BlockCard）

```markdown
# 阻塞卡 BlockCard-{任务编号}
发现问题时间：YYYY-MM-DD HH:mm

## 任务编号
TC-XXX-niuniu

## 问题描述
[具体问题]

## 已尝试的解决方式
[列出]

## 需要 PM 支持的内容
[具体说明]

## 预计阻塞时间
X 小时
```
