## 任务内容

### 1. 项目结构初始化
按「妞妞个人开发计划 V1.2」目录规范初始化 Go 项目：

```
backend/
├── cmd/server/main.go
├── internal/
│   ├── config/          ← 配置加载
│   ├── handler/         ← HTTP handler（Gin 路由处理）
│   ├── middleware/      ← 中间件（JWT/CORS/日志）
│   ├── model/           ← 数据模型
│   ├── repository/      ← 数据访问层
│   ├── service/         ← 业务逻辑层
│   ├── router/          ← 路由注册
│   ├── websocket/       ← WebSocket 实时通信
│   └── strategy/        ← 策略执行引擎
├── pkg/
│   ├── response/        ← 统一响应格式
│   ├── jwt/             ← JWT 工具
│   └── logger/          ← 日志工具
├── migrations/         ← 数据库迁移脚本
├── go.mod
└── Dockerfile
```

### 2. 数据库 & Redis 连接
- 连接 PostgreSQL（biz 库，biz_ 表前缀）
- 连接 Redis（biz: 前缀缓存）
- 使用 GORM 做 ORM

### 3. 实现健康检查接口
```
GET /api/v1/health
响应：{ "code": 0, "data": { "status": "ok" } }
```

### 4. WebSocket 骨架（预置配置）
```
WS /api/v1/ws
```
预置心跳间隔10秒 / 断开自动重连最多10次间隔2秒 / 推送延迟≤200ms

### 5. 输出 API 草案 v0.1
包含：统一响应格式（{code, message, data}）、错误码规范（1xx~5xx）、全部路由列表
