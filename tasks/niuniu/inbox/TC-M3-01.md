## 任务内容

### 1. 用户注册
```
POST /api/v1/auth/register
Body: { username, password, email, role? }
响应：{ code: 0, data: { user_id, token } }
```
- 密码 bcrypt 加密存储
- 用户名唯一性校验
- 缺省角色：user

### 2. 用户登录
```
POST /api/v1/auth/login
Body: { username, password }
响应：{ code: 0, data: { token, user_id } }
```
- JWT Token（有效期 7 天）
- 登录日志记录

### 3. JWT 鉴权中间件
- 所有 /api/v1/* 接口（除 /auth/* 外）需携带 Bearer Token
- Token 失效 / 缺失返回 401

### 4. 当前用户接口
```
GET /api/v1/auth/me
响应：{ code: 0, data: { user_id, username, email, role } }
```

### 5. 用户画像
```
GET  /api/v1/user/profile         ← 获取
PUT  /api/v1/user/profile         ← 更新（username/email/avatar）
```
- 用户画像表：biz_user_profile

### 6. 权限控制
- 角色：admin / user
- admin 可查看所有用户、修改权限
- user 仅操作自己的数据

## 交付物
- [ ] POST /api/v1/auth/register 可正常注册（密码加密）
- [ ] POST /api/v1/auth/login 返回 JWT Token
- [ ] GET /api/v1/auth/me 返回当前用户信息
- [ ] GET /api/v1/user/profile 可获取用户画像
- [ ] PUT /api/v1/user/profile 可更新画像
- [ ] 无 Token 请求 /api/v1/user/profile → 401

## 审核点
1. 注册后 Token 有效，可正常登录
2. Token 过期或伪造 → 401
3. user 角色无法访问 admin 接口
4. biz_user 表有记录，密码为加密值（非明文）
