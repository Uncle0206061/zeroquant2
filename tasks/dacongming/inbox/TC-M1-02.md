## 任务内容

### 1. 项目初始化
按「大聪明个人开发计划 V1.2」目录规范初始化 Vue3 + Vite 项目：

```
frontend/
├── public/
├── src/
│   ├── api/             ← API 请求封装（统一封装 /api/v1/ 和 /data/v1/）
│   ├── assets/          ← 静态资源
│   ├── components/     ← 公共组件（PC + 移动端共用）
│   ├── composables/     ← 组合式函数

│   ├── router/         ← 路由配置（含权限守卫）
│   ├── stores/         ← Pinia 状态管理
│   ├── utils/          ← 工具函数

│   ├── App.vue
│   └── main.ts
├── index.html
├── vite.config.ts
├── tsconfig.json
├── package.json
└── Dockerfile
```

### 2. 核心依赖安装
Axios / Pinia / VueRouter / Element Plus / Vant / ECharts / WebSocket

### 3. API 统一封装（Axios 拦截器）
- 请求拦截器：自动附加 JWT 到所有请求头
- 响应拦截器：统一处理 {code, message, data} 格式
- baseURL 配置：/api/v1/（Go 后端）和 /data/v1/（数据服务）

### 4. WebSocket 全局封装

- 心跳间隔：10 秒

- 推送延迟要求：≤200ms
- 全局唯一实例，跨页面复用

### 5. 路由守卫
未登录跳转 /login，登录后正常访问

### 6. 确认前端 API 统一封装方案
