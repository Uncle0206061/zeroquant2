## 任务内容

### 1. 策略编辑器形态
形态：表单勾选 + 填写（参考需求文档 REQ-S004 策略编辑器）
- 左侧：因子选择树（财务指标/技术指标/板块概念）
- 右侧：参数填写表单（数值、日期、开关等）
- 底部：预览 JSON + 保存/提交按钮

### 2. 选股条件（因子面板）
| 分类 | 因子 | 参数 |
|------|------|------|
| 财务指标 | 市盈率 PE | min/max |
| 财务指标 | 市净率 PB | min/max |
| 财务指标 | 市值 | min/max |
| 技术指标 | MA 均线 | 周期、突破类型 |
| 技术指标 | MACD | 快线/慢线/信号线参数 |
| 技术指标 | RSI | 周期、超买超卖阈值 |
| 板块概念 | 行业板块 | 板块代码列表 |
| 板块概念 | 概念题材 | 题材列表 |

### 3. 择时条件（条件面板）
- 买入时机：MA 金叉/死叉、MACD 背离、RSI 超卖等
- 止损/止盈：固定百分比、跟踪止损
- 风控参数：最大仓位、单笔上限、持仓周期

### 4. 输出格式
前端编辑器输出的策略 JSON 结构：
```json
{
  "name": "策略名称",
  "stock_filter": { "pe": { "min": 0, "max": 30 }, "sector": ["银行"] },
  "timing": { "type": "ma_cross", "period": 20 },
  "risk": { "max_position": 0.3, "stop_loss": 0.05 }
}
```
与后端 biz_strategy 表 JSON 字段格式一致。

### 5. 编辑器交互
- 因子树可多选（checkbox）
- 选中因子右侧展开参数表单
- 实时预览输出 JSON
- 保存草稿 / 提交执行两个按钮
- 提交后 WebSocket 接收回测进度推送
