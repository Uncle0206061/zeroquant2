## 移动端功能详情

### 横屏检测
- resize + orientationchange 双事件监听
- 横屏时全屏遮罩 + 📱 旋转动画 + "请使用竖屏浏览"提示

### 离线缓存
- useNetworkStatus composable（online/offline 事件）
- Pinia persist：回测结果持久化到 localStorage
- 离线时：显示 NoticeBar 提示、禁用刷新/下单/撤单按钮、展示最后缓存数据

### 移动端持仓页面
- 深色渐变概览卡（总市值+总盈亏+持仓数+胜率）
- 卡片式持仓列表（名称+代码+现价+成本+盈亏%+盈亏额）
- 底部弹出详情 Popup（7 个字段）
- 下拉刷新

### 移动端订单页面
- Vant Tabs 筛选（全部/待成交/已成交/已撤单）
- 卡片式订单列表（方向+价格+数量+成交量+时间+状态标签）
- pending 订单可撤单（showConfirmDialog）
- 下拉刷新

### 移动端回测页面
- 简化表单（Field 输入）
- 6 个指标卡片网格
- 3 个 ECharts 图表（资金曲线/回撤曲线/收益分布）
- 成交记录列表（最多展示 20 条 + "还有 N 条"提示）
- Pinia 持久化：PC 端回测结果移动端可看
