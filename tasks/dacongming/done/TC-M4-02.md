## Part A：持仓页面（/position）

### 新增文件
| 文件 | 说明 |
|------|------|
| `src/composables/usePosition.ts` | 持仓数据 hook（加载/实时价格/WS/汇总） |
| `src/views/PositionView.vue` | 持仓页面 |

### 功能详情
- **数据加载**：GET /api/v1/position → 逐只获取 /data/v1/market/{code} 实时价格
- **盈亏计算**：盈亏金额 = (现价 - 成本) × 数量，盈亏比例 = 盈亏 / (成本 × 数量)
- **颜色语义**：A 股习惯 — 盈红（#F56C6C）亏绿（#67C23A）
- **排序**：按盈亏比例降序
- **占比进度条**：每只股票市值占总市值比例（颜色跟随盈亏）
- **汇总区**：持仓数量 / 总市值 / 总盈亏 / 胜率
- **定时刷新**：30 秒自动刷新价格
- **WebSocket**：position_update 全量更新持仓

| `src/views/OrdersView.vue` | 订单列表页面 |

### 功能详情
- **数据加载**：GET /api/v1/order/list（page=1, page_size=200）
- **筛选**：方向（买入/卖出）+ 状态（5 种）+ 重置
- **状态标签颜色**：pending=黄 / filled=绿 / partial_filled=蓝 / cancelled=灰 / rejected=红
- **撤单**：pending 状态订单可撤，二次确认弹窗
- **成交明细**：点击行弹出 Dialog，展示完整订单信息（10 个字段）
- **分页**：前端分页，每页 20 条
- **WebSocket**：order_update 自动合并更新（新增或修改）
