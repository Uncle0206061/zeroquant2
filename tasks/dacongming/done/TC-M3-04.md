### 功能详情

**因子选择树（左侧面板）：**
- 财务指标：PE / PB / 市值 / ROE / 营收增长率
- 技术指标：MA 均线 / MACD / RSI / 布林带 / 成交量
- 板块概念：行业板块 / 概念题材
- 多选 checkbox，选/取消自动联动参数表单

**参数表单（中间面板）：**
- 数值型：`el-input-number`，支持 min/max/step/precision/unit
- 选择型：`el-select`，支持 options
- 标签型：`el-check-tag`，支持 toggle 多选
- 择时条件：7 种时机（MA金叉/死叉、MACD背离、RSI超卖等）
- 风控参数：最大仓位（滑块）、止损/止盈（百分比）、最大持仓天数

**JSON 预览（右侧面板）：**
- 实时响应式计算 `computed`
- 输出格式与后端 `biz_strategy` JSON 一致
- 一键复制功能

**操作按钮：**
- 重置 → 清空全部表单
- 保存草稿 → `POST /api/v1/strategy/save`
- 提交执行 → `POST /api/v1/strategy/submit` + WebSocket 监听 `backtest_progress`
