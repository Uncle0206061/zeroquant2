## Part A：回测页面

### 新增文件
| 文件 | 说明 |
|------|------|
| `src/composables/useBacktest.ts` | 回测逻辑 hook（发起/进度/结果/图表渲染） |
| `src/utils/echarts.ts` | ECharts 按需引入（Line+Bar+Tooltip+Grid） |
| `src/views/BacktestView.vue` | 回测页面（表单+指标+图表+成交表） |

### 功能详情

**发起回测表单：**
- 股票代码 / 开始结束日期 / 初始资金
- 进度条（WebSocket backtest_progress 实时更新）
- 备用轮询（60 次 × 2 秒间隔）

**核心指标卡片（6 个）：**
- 年化收益率 / 最大回撤 / 夏普比率 / 胜率 / 盈亏比 / 总交易次数

**ECharts 图表（3 个）：**
- 资金曲线（折线 + 面积填充）
- 回撤曲线（折线 + 红色面积）
- 收益分布（柱状图，涨红跌绿）

**成交记录表：**
- 时间 / 代码 / 名称 / 方向（买红卖绿标签）/ 价格 / 数量 / 金额

## Part B：消息提醒系统

### 新增文件
| 文件 | 说明 |
|------|------|
| `src/components/NotificationCenter.vue` | 全局通知组件（铃铛+面板+浮窗+声音） |

### 修改文件
| 文件 | 改动 |
|------|------|
| `src/layouts/MainLayout.vue` | 集成 NotificationCenter 组件 |

### 功能详情

**通知类型（5 种）：**
| 类型 | 标签 | 触发 |
|------|------|------|
| order_filled | 成交 ✅ | WS order_update (status=filled) |
| order_rejected | 拒绝 ❌ | WS order_update (status=rejected) |
| backtest_complete | 回测完成 🔵 | WS backtest_result |
| position_alert | 持仓预警 ⚠️ | WS position_update (alert) |
| system | 系统 ⚪ | WS system_alert |

**交互：**
- 铃铛角标显示未读数量（localStorage 持久化）
- 未读时有摇晃动画
- 点击展开通知面板（全部已读 / 清除）
- 右下角浮窗提醒，5 秒自动消失，支持多条同时显示
- Web Audio API 提示音（无需外部文件），开关控制
- 通知保留 24 小时（localStorage 自动清理）
