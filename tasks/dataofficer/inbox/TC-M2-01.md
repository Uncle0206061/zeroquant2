## 任务内容

### 1. Level-2 五档盘口采集
实现 /data/v1/orderbook/{stock_code} 接口：
- 五档行情（买1~买5，卖1~卖5）
- 实时数据源接入（akshare / Tushare Pro / Level-2）
- 主数据源断开时 1 秒内自动切换备用源
- 全部源失效时返回错误，不返回脏数据

### 2. 逐笔成交采集
实现 /data/v1/tick/{stock_code} 接口：
- 逐笔成交数据（时间、价格、成交量、方向）
- 采集任务异常自动重试

### 3. K 线数据
实现 /data/v1/kline/ 接口：
- 支持日K / 60分钟K / 30分钟K / 15分钟K / 5分钟K / 1分钟K
- 数据源：akshare / Tushare Pro
- 数据必须落盘 + 缓存双写

### 4. 实时行情
实现 /data/v1/market/{stock_code} 接口：
- 当前价格、涨跌、成交量、成交额
- 板块信息 /data/v1/sector/{sector_code}

### 5. 缓存策略
- 盘口数据：Redis 缓存，TTL 5s
- 逐笔数据：保存 90 天
- K 线数据：永久存储（data_kline 表）
- 降级机制：数据源故障时返回缓存数据 + 标记 stale
