## 任务内容

### 1. 数据质量保障
- 盘口数据：akshare / Tushare Pro / Level-2 三源可用
- 数据校验：日终对账（成交额、成交量与交易所数据偏差 <1%）
- 异常数据：自动标记（数据偏差 >1% 写入 data_anomaly_log）

### 2. 数据监控与告警
- 采集失败 → 自动重试（最多 3 次）→ 失败告警（WebSocket → 管理员）
- 数据延迟：超过 5 分钟未更新 → 告警
- Redis 缓存命中率 <80% → 告警

### 3. 历史数据归档
- K 线数据（data_kline）：永久存储
- 逐笔数据（data_tick）：90 天滚动删除
- 归档策略：每日凌晨 3 点执行（delete where timestamp < now() - 90d）

### 4. Docker 部署
- data/Dockerfile（Python 最小化镜像）
- docker-compose.yml 整合三端（Go/Python/前端/PG/Redis）
- 启动顺序：PostgreSQL → Redis → Python 数据服务 → Go 后端 → 前端

### 5. 文档与测试
- API 文档：Swagger（FastAPI auto-generated）
- 数据字典：docs/data-dictionary.md（含表结构、字段说明）
- 接口测试覆盖：所有 /data/v1/ 接口（pytest）
