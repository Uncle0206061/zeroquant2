## 修复内容

### 1. CORS 生产风险
- **问题**: CORS 配置硬编码，生产环境风险
- **修复**:
  - `config.go`: 添加 `CORSAllowedOrigins` 配置项
  - `middleware.go`: CORS 中间件从配置读取
  - `config.yaml`: 添加 `cors_allowed_origins` 配置

### 2. 模拟回测重复执行
- **问题**: Python 服务成功响应后仍调用 `simulateBacktestComplete`，导致重复推送
- **修复**: `strategy_service.go` 只在调用失败时模拟回测，成功后等待 Python 回调

### 3. WebSocket CheckOrigin 风险
- **问题**: CheckOrigin 返回 true，太宽松
- **修复**: `websocket.go` 从配置读取允许的 origin 列表

### 4. RateLimit 内存泄漏
- **问题**: 内存 map 只增不减
- **修复**: `middleware.go` 使用 Redis 限流，Redis 不可用时降级到内存限流

### 5. Watchlist 表缺失
- **问题**: model 定义了 Watchlist 但未在 AutoMigrate 中创建
- **修复**: `main.go` 添加 `&model.Watchlist{}` 到 AutoMigrate
