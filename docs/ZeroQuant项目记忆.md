## 七、策略编辑器（v2.2 确定）

**形态：** 表单勾选 + 填写

**流程：** 新建 → 选择因子 → 配置条件 → 逻辑组合 → 保存 → 回测 → 上线运行

**版本管理：** 支持策略版本管理，可回滚到历史版本

**支持因子：**
- 基础行情：最新价、涨跌幅、成交量、成交额、换手率、量比、涨停价、跌停价
- 盘口因子：买一/卖一量、五档总量、委比、委差、盘口厚度
- 技术因子：MA5/MA10/MA20、MACD 金叉/死叉、KDJ、RSI、BOLL 上下轨
- 资金因子：主力净流入、超大单净流入

**规则组合：** AND / OR / NOT 嵌套，最多 10 层条件

**触发方式：** 实时触发、定时触发、K 线闭合触发

## 十、实盘规则（v2.2 量化）

| 规则 | 说明 |
|------|------|
| 对接方式 | 支持券商开放平台接口（优先：东方财富/同花顺/券商官方） |
| 账户隔离 | 模拟盘与实盘完全隔离，独立账户、独立策略开关 |
| 默认状态 | 实盘默认关闭，必须手动开启 |
| 紧急暂停 | 提供一键暂停所有实盘策略按钮 |
| 风控暂停 | 触发风控阈值后自动暂停实盘，需手动恢复 |
| 异常冻结 | 异常自动冻结交易，禁止自动恢复 |

## 十一、异常处理（v2.2 量化）

| 场景 | 处理方式 |
|------|---------|
| 主数据源断开 | 1 秒内自动切换备用源 |
| 全部源失效 | 停止交易，推送告警，不执行任何策略 |
| 策略执行失败 | 不重试，标记失败并推送告警 |
| 服务崩溃 | 容器自动重启，重启后恢复策略状态 |
| 数据库异常 | 熔断保护，停止写入，保护数据 |

## 十五、风险与应对（v2.2 新增）

### 技术风险

| 风险 | 应对方式 |
|------|----------|
| Level 2 数据源不稳 | 多源备份，1 秒自动切换 |
| WebSocket 压力 | Redis 发布订阅 + 连接池 |
| 数据库异常 | 熔断保护，停止写入 |

### 业务风险

| 风险 | 应对方式 |
|------|----------|
| 策略执行时效 | 模拟盘充分测试后上线 |
| 用户数据安全 | 敏感信息脱敏 + AES 加密 |
| 异常交易 | 自动冻结，禁止自动恢复 |
